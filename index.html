Hammerspace
Data Storage Platform

Unified Global Data Fabric for AI Anywhere

Hammerspace is a global file system that lets your AI workloads access data wherever it resides—on-premises, in other data centers, or in the cloud—without needing to move or copy that data manually. It presents all your storage as a single, unified namespace, regardless of where the physical data is stored.

Combined with Hitachi VSP One Block storage, it ensures that data is always available, resilient, and space-efficient—making it ideal for modern, distributed AI pipelines.


Key Benefits with Hitachi iQ:

  • 100% data availability SLA
  • 4:1 data reduction, no questions asked
  • AI-ready data fabric across edge, core, and multi-cloud
  • Real-time, zero-copy GPU access to remote datasets
  • Global namespace with high-speed metadata acceleration
  • Dedicated metadata engine for fast data placement and retrieval
  • Automatic data orchestration & movement based on compute proximity
  • Ideal for RAG pipelines, LLM tuning, and multi-tenant AI

HCSF
Parallel File System

Accelerated Storage for AI Workloads

Hitachi Content Software for File (HCSF) is a high-performance parallel, scale-out file system designed to deliver high throughput and low latency—ideal for AI and machine learning workloads. It is powered by the Weka® file system, providing the performance and flexibility needed to support demanding AI pipelines at enterprise scale.

In the Hitachi iQ platform, HCSF feeds large volumes of data into AI models quickly and efficiently. It supports POSIX-compliant file access, integrates with NVIDIA GPUDirect for direct storage-to-GPU communication, and avoids traditional I/O bottlenecks that slow down training or inferencing jobs.

When paired with Hitachi storage hardware and DGX or HGX systems, HCSF enables AI pipelines to operate at full speed—from data ingestion to model output.


Key Benefits:

  • Ultra-low latency for AI/ML workloads
  • Massive parallel throughput for large datasets
  • Native POSIX file system with multiprotocol support
  • Direct GPU access via NVIDIA GPUDirect Storage
  • Scales linearly with additional nodes
  • Optimized for AI training and inference
